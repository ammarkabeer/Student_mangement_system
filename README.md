# Student Management System

This is a simple C++ console application for managing student records. The system allows you to:

- Add new students (ID, name, age)
- Display all students
- Search for a student by ID
- Delete a student by ID

The application uses basic C++ features such as classes, vectors, and standard input/output.

## Usage

1. Compile the code using a C++ compiler.
2. Run the executable.
3. Follow the on-screen menu to manage student records.
